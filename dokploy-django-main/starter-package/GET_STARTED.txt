🎯 QUICK START (3 STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  SETUP TRAEFIK (One time, on VPS)
    
    cd scripts/
    ./traefik-setup.sh


2️⃣  READ THE GUIDE

    Open: docs/QUICK_START.md
    Follow step-by-step instructions


3️⃣  USE THE TEMPLATES

    templates/.env.production.template    → Copy to Dokploy UI
    templates/nginx-template.conf         → Setup nginx
    
    Then deploy!
